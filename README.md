# Portfólio de Projetos — Engenharia de Software

Bem-vindo ao meu repositório de projetos!  
Aqui você encontrará uma coleção organizada dos trabalhos que desenvolvi ao longo da minha formação em **Engenharia de Software**, incluindo:

- Algoritmos em C  
- Automações e sistemas em Python  
- Modelagem UML e BPMN  

Este repositório reúne meus primeiros passos na programação, estruturação de sistemas e modelagem de processos. Cada pasta representa um conjunto de projetos independentes, cada um com seu próprio README detalhado.

---

## Estrutura do Repositório

### **1. Algoritmo de Operações (C)**
Pasta contendo exercícios e algoritmos essenciais realizados em linguagem **C**, incluindo:
- operações aritméticas  
- comparações numéricas  
- soma de valores  
- cálculo de salário líquido  
- controle de vendas  

**Tecnologias:** C, GCC, Linux  
**README detalhado dentro da pasta**

### **2. Automações e Modelos (Python)**
Repositório com sistemas simples e automações:

 **Tecnologias:** Python, Pandas, NumPy, Scikit-Learn  
 **README detalhado dentro da pasta**

### **3. Diagramas UML e BPMN**
Contém diagramas produzidos para análise e modelagem de sistemas:
