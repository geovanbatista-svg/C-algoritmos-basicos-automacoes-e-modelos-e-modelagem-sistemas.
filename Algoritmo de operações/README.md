# Algoritmo de Operações

Este repositório contém uma coleção de algoritmos fundamentais desenvolvidos em linguagem C.  
Os exercícios fazem parte da minha formação em lógica de programação e estrutura básica da linguagem.

##  Funcionalidades

Os programas implementam lógicas essenciais, incluindo:

- Operações aritméticas básicas  
- Comparação entre três números fornecidos pelo usuário  
- Soma de valores inteiros digitados  
- Registro de vendas diárias  
- Cálculo de salário líquido  

Cada script foi criado para praticar:
- Entrada e saída de dados
- Condicionais (`if / else`)
- Loops
- Variáveis e operadores
- Estruturação de pequenos programas

##  Tecnologias utilizadas
- C (GCC)
- Linux / Ubuntu
- Git & GitHub
